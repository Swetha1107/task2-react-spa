# Product Management System (MERN Stack)

## 📌 Description
This is a Product Management System built using MERN Stack Internship Task 2.

Frontend is built using React.js and it connects to backend REST API built using Node.js, Express.js, and MongoDB.

---

## 🚀 Running URLs

### Frontend (React)
http://localhost:5173

### Backend (API Server)
http://localhost:5000

---

## ✨ Features
- Add Product
- View Products
- Update Product
- Delete Product

---

## 🛠️ Technologies Used

### Frontend
- React.js
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

---

## 🔗 API Endpoints

- GET /api/products → Get all products
- POST /api/products → Add product
- PUT /api/products/:id → Update product
- DELETE /api/products/:id → Delete product
